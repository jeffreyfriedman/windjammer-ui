# Windjammer UI v0.1.0 - First Public Release

## Overview

Windjammer UI is a cross-platform UI framework for building reactive desktop applications with the Windjammer programming language.

## Features

- **Reactive State Management** - Signal-based reactivity inspired by Solid.js
- **30+ UI Components** - Rich widget library with builder pattern API
- **Automatic Re-rendering** - UI updates automatically when Signals change
- **Zero Configuration** - Compiler handles dependency management
- **Desktop Ready** - Native performance on Linux, macOS, Windows

## Getting Started

```windjammer
use std::ui::*

@export
fn start() {
    let count = Signal::new(0)
    let count_ref = count.clone()
    
    let ui = Container::new()
        .child(Button::new("Click me")
            .on_click(move || count_ref.set(count_ref.get() + 1)))
    
    App::new("My App", ui.to_vnode()).run()
}
```
